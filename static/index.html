Quick Start

Get Started in Minutes

Install, configure, and run your first medical summary with a single command.

bash Installation & Usage 
# 1. Clone the repository
git clone https://github.com/your-org/medical-summary-builder

# 2. Install with uv (recommended)
uv sync

# 3. Configure your API token
cp .env.example .env
# Edit .env and add your AI_BUILDER_TOKEN

# 4. Run the pipeline
uv run python -m medical_summary_builder.main \
  --pdf "data/Medical File.pdf" \
  --template "docs/summary_template.docx" \
  --output "output/summary.docx"
