    How It Works
    
  
    5-Stage Agent Pipeline
    
  
    Each stage is a specialised AI agent that builds on the previous one's output.
    
  
    
    
    
      
    📑
    
      
    1 · Extraction
    
      
    Parses every page of the PDF with pdfplumber. SHA-256 caching means unchanged files are never re-parsed.
    
    
    
    
    
      
    🧠
    
      
    2 · Analysis
    
      
    LLM extracts demographics, conditions, and a chronological visit timeline — section by section to stay within token limits.
    
    
    
    
    
      
    ✅
    
      
    3 · Validation
    
      
    Fuzzy matching cross-checks each visit row against its cited PDF page. Hallucinations are corrected or removed.
    
    
    
    
    
      
    📝
    
      
    4 · Report
    
      
    Fills every {{PLACEHOLDER}} in the Word template and writes the sorted visit timeline table.
    
    
    
    
    
      
    ⬇️
    
      
    5 · Download
    
      
    Receive a polished .docx ready for review — no login, no storage, files are deleted after download.
